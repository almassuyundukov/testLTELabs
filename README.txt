Инструкция по работе с системой записи пациентов на прием:
1. В postgreSQL нужно создать таблицу с названием test-LTE-labs
2. В application.yaml указать данные для входа от вашей базы данных (логин, пароль)
3. При первом запуске свойство ddl-auto указать как create, далее validate.
4. Реализованный запрос на SOAP-сервис находится в папке example. Он создает 10 записей на дату 2 сентября 2022 года с периодом в 30 минут, начиная с 7:30.
5. Реализованные примеры запросов на RESTful находятся в папке example в формате JSON.
